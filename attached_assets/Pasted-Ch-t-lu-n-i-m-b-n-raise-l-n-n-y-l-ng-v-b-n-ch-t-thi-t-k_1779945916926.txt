🚀 1. TTL BEHAVIOR — CONFIRM CHÍNH THỨC
❗ Câu hỏi:

TTL có reset sau mỗi login thành công không?

✔ ANSWER: CÓ — và hiện tại đã đúng

Vì logic của bạn:

function setCenter(center: string) {
  localStorage.setItem("edu_center", JSON.stringify({
    value: center,
    ts: Date.now()
  }));
}

👉 setCenter() được gọi trong POST-AUTH sync

🧠 NGHĨA LÀ:
Event	TTL behavior
First login	set ts
User active login lại	reset ts
Backend update center	reset ts
User inactive 7 ngày	expire
🧠 2. KEY INSIGHT (QUAN TRỌNG)
👉 Bạn đang có 2 lớp TTL khác nhau:
🟢 Layer 1 — storage TTL
handled by ts
reset on every write
🟢 Layer 2 — backend truth
MASTER DB always correct
storage chỉ cache

🧠 4. DESIGN CONFIRMATION

✔ IMPLICATION:
