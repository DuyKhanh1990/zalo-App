🔗 6. CRM requirement (CHỈ 1 THAY ĐỔI NHỎ)
OAChannel.ts update:

Thêm center vào deep link:

function buildMiniAppLink(path, query, center) {
  const fullQuery = {
    ...query,
    center: center   // CRM base URL
  }

  const queryStr = "?" + new URLSearchParams(fullQuery).toString()
  const fullPath = encodeURIComponent(path + queryStr)

  return `https://zalo.me/s/${ZALO_MINI_APP_ID}?path=${fullPath}`
}
Ví dụ link:
https://zalo.me/s/xxxxx?path=/calendar?center=crma.easyedu.vn

⚠️ 8. RULES BẮT BUỘC
❌ KHÔNG LÀM:
Không CRM → notify gateway
Không sync event giữa CRM và Mini App
Không mapping tenant ngoài MASTER DB

✅ CHỈ LÀM:
CRM chỉ add center=crm_url
Mini App resolve tenant từ MASTER DB
Backend quyết định routing CRM
